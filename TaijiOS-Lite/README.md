# TaijiOS Lite v1.4.0

## 全球第一个带自进化能力的AI认知军师操作系统

**不是聊天机器人。不是AI助手。是你的认知军师——像诸葛亮对刘备，一针见血，越用越懂你。**

每一个TaijiOS实例就是一个独立的智能体(Agent)。每个Agent有自己的经验结晶、易经卦象、认知地图。所有Agent之间可以互相学习、共享经验、交叉验证。**网络越大，每个人越聪明。**

## 核心能力

- **跨会话记忆** — 关了再开，军师记得你是谁、上次聊到哪、你的所有经验

- **13种AI模型** — DeepSeek/R1/GPT/Claude/通义/GLM/豆包/Kimi...国内国外随便选
- **傻瓜模式** — 双击exe，选个1，粘贴Key，开聊。30秒搞定

```

  ├─ 7. 贡献积分系统 ──── 8种积分来源，5个等级
  └─ 8. 生态制度 ────────  Agent网络 + 16成就 + 5角色
```

## 快速开始

### 方式一：下载exe（推荐，不需要Python）

从 [Releases](../../releases) 下载 `TaijiOS-Lite.exe`（19MB），双击运行：

3. 有ICI文件(.docx)放同目录自动识别，没有就30秒快速建档
4. 开聊

### 方式二：从源码运行

```bash
pip install python-docx openai python-dotenv
python taijios.py
```

### 安全防护
- **注入拦截** — 导入.taiji文件时自动过滤 prompt 注入攻击
- **内容截断** — 单条规则最长100字，单次最多导入20条
- **原子写入** — 崩溃不丢数据，JSON损坏自动恢复
- **冷却防刷** — 积分有冷却时间，防止刷分
- **API重试** — 网络抖动自动重试一次

## Agent互学机制

```
Agent A 对话进化 → 结晶 → 导出 .taiji（v2: 结晶+易经+灵魂，已脱敏）
                                ↓
Agent B 导入 → 规则融入认知 → 验证 → 置信度上升
                                ↓
Agent C 导入A+B → 交叉验证 → 全网经验质量越来越高
                                ↓
新Agent加入 → 直接继承最佳经验 → 起点更高
```

**.taiji v2 经验包** 不只是规则列表，还携带：
- 来源Agent的易经卦象快照（当时的状态）
- 来源Agent的认知模式（匿名脱敏的五维分析）
- 验证计数（被多少Agent采纳过）

## 易经卦象引擎

| 爻位 | 维度 | 阳(积极) | 阴(消极) |
|------|------|----------|----------|
| 初爻 | 情绪基底 | 情绪稳定 | 焦虑挫败 |
| 二爻 | 行动力 | 有目标在行动 | 迷茫纠结 |
| 三爻 | 认知清晰度 | 自我认知清 | 混沌不清 |
| 四爻 | 资源状态 | 有资源支持 | 匮乏缺乏 |
| 五爻 | 方向感 | 方向明确 | 摇摆不定 |
| 上爻 | 满意度 | 正面为主 | 负面为主 |

16种核心军师策略：乾(进攻)、坤(防守)、屯(启动)、蒙(启蒙)、需(等待)、讼(调解)、师(统帅)、比(联盟)、观(点破)、剥(止损)、复(复苏)、困(破局)、渐(稳进)、既济(居安)、未济(冲刺)、涣(聚焦)

### 变爻推演（v1.3.2 新增）

每3轮对话，军师自动推演你的走势：

```text
  ┌─────────────────────────────────────
  │ 军师推演 —— 困中寻源
  │
  │ 当前：泽水困 ⚋⚋⚊⚊⚊⚋
  │ 变爻：初爻(情绪基底)阴→阳, 三爻(认知)阴→阳
  │ 变卦：泽水困 → 水风井 ⚊⚊⚊⚋⚊⚊
  │
  │ 推演：你觉得被困住，但答案不在外面，在你自己身上。
  │
  │ 建议：列出你已有的所有资源，重新审视哪些能组合出新价值。
  └─────────────────────────────────────
```

- **变爻检测** — 哪些维度在发生变化（情绪/行动力/认知/资源/方向/满意度）
- **变卦推算** — 当前卦 + 变爻 → 未来卦象
- **18组精确推演** — 困→井(困中寻源)、坤→复(否极泰来)、既济→未济(盛转危)...
- **自动触发** — 不需要操作，聊3轮军师自己推演

## 什么是ICI

ICI = Individual Cognitive Identity = 个体认知身份标识

把一个人拆成5个维度（位置、本事、钱财、野心、口碑），形成完整的认知档案。TaijiOS读取这个档案，激活为你的专属认知军师。

没有ICI文件也能用——7个问题30秒快速建档。
